# YUANFU Portfolio

## 风格

Web3/Cyberpunk 风格，紫色 (#aa3bff) + 青色 (#00d4ff) 霓虹主题，玻璃拟态 UI，终端打字动画，响应式布局。

## 技能

- **Media Player** - Ijkplayer, FFmpeg, DASH, Metal, OpenGL ES
- **Cross-Platform** - React Native, Hummer, Dimina, TypeScript, WebView
- **Hardware & Embedded** - VideoToolBox, Metal, Remote-Control, RTMP, Motion

## 技术栈

Vue 3 + TypeScript + Vite，使用 Vue Router 和 CSS 自定义属性。

## 页面

- `/` - 主页
- `/lobster.html` - 龙虾介绍页

## 开发

```bash
yarn install
yarn dev
yarn build
```

## 部署

构建产物在 `dist/` 目录，直接用 nginx serve 即可。
